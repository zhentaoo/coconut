# Coconut

## 网站的技术结构
* nodejs(express) + mongodb
* 网站地址：http://www.zhentaoo.com
* 服务器在阿里云，网站已备案.

<br><br>

## 网站的几个基础功能：
  * 第三方登录接入
    * weibo 登录接入

  * 在线聊天功能
    * 登录后可以在公共频道发言

  * 博客功能
    * 后台文章发布
    * 前台文章倒叙展示，limit 15
    * 前台文章分类：类别，每个类别下的列表
